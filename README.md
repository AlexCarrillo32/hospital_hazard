# Hospital Hazard Tracking System

A modern React-based application for tracking and managing hospital hazards, safety incidents, and compliance reporting.

## Features

- Real-time hazard tracking and reporting
- User-friendly incident logging
- Dashboard for hazard analytics
- Compliance reporting tools

## Tech Stack

- **Frontend**: React 19.1
- **Build Tool**: Vite 7.1
- **Styling**: CSS Modules
- **Linting**: ESLint 9.x

## Getting Started

### Prerequisites

- Node.js 18+ and npm

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) to view the app in development mode.

### Build

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```
